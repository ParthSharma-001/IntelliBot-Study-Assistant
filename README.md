# IntelliBot: Your Personalized AI Study Assistant

**by Parth Sharma**

IntelliBot is a web-based AI study assistant designed to empower students with intelligent support and streamlined academic management.

## ✨ Key Features

* 🧠 **Intelligent Chatbot:** Ask academic questions and get instant, contextual answers powered by Google Gemini 2.0 Flash API.
    * *Hybrid AI:* Handles app-specific commands locally, reserves Gemini for complex queries.
    * *Enhanced Formatting:* Uses **Marked.js** for clear, readable chatbot responses.
* 📈 **Daily Progress Tracker:** Monitor questions answered, daily goals, and maintain your "study streak."
* ✅ **Task Management System:** Easily add, track, and complete your study tasks.
* ⏰ **Study Reminders:** Set timely browser notifications for study sessions and deadlines.
* 🖥️ **Engaging UI:** Modern, responsive design with an immersive visual experience.
    * *Dynamic Background:* Features an animated 3D particle background powered by **Three.js**.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Libraries:** Marked.js, Three.js
* **AI Integration:** Google Gemini 2.0 Flash API
